# Compiler
Compiling with dodgey GUIs no more! Compile from APK, or source to a windows EXE or a website with iOS support!

## Requirements
- Python 3.11 or higher

## How to use:
### 1. Setup
Run this.
```
python -m pip install -r requirements.txt
```
### 2. Formatter!
1. Run the following command:
```
python formatter.py
```
2. Input the conversion option you want, and then do as it follows!
3. It will successfully output to the `output` folder!
